# Вклад работы

Работа делает три вещи.

1. Строит HANK-основанную локальную среду, где полное распределение домохозяйств сжимается до нескольких статистик, значимых для трансмиссии ставки.
2. Измеряет ценность распределительной информации через снижение функции потерь и сокращение разрыва до полной информации.
3. Показывает, в каких средах агрегатная информация почти достаточна, а в каких распределительные статистики дают дополнительную ценность.

Главная интерпретация: распределительная информация ценна не сама по себе, а тогда, когда она улучшает оценку будущей силы трансмиссии процентной ставки.

Работа не утверждает, что решает полную задачу оптимальной политики в HANK-модели. Она оценивает ценность разных низкоразмерных информационных состояний.
