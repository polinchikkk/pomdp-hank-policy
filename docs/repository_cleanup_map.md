# Карта очистки репозитория

## Архивные ветки

Старые материалы сохранены в архивных ветках:

```text
archive/hank-legacy
archive/pre-hank-only-main
archive/stage6-before-information-state-redesign-20260430
archive/markov-regime-information-state-before-distribution-voi-20260501
```

Первые две ветки хранят более ранние состояния HANK-ядра. Третья ветка хранит прежние эксперименты шестого этапа. Четвертая ветка хранит промежуточную линию про информационные состояния в среде со скрытыми марковскими режимами.

## Что осталось в основной ветке

В основной ветке оставлена только новая линия:

```text
HANK / метод последовательностей
+ неполная наблюдаемость
+ ценность распределительной информации
```

Текущие рабочие папки:

- `hank/`;
- `state_space/`;
- `policy/`;
- `experiments/`;
- `config/`;
- `docs/`;
- `article/`.

## Что удалено из основной ветки

Из основной ветки убраны:

- старый блок обучаемых правил;
- среда со скрытыми марковскими режимами;
- старые скрипты частичной информации и режимного обучения;
- старые выходные таблицы и графики.

Эти материалы не уничтожены: они доступны в архивных ветках.
