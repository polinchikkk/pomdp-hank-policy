\section{HANK-модель}

Полное состояние HANK-экономики обозначается через
\[
X_t^{HANK}
=
\left(
\Gamma_t,
A_t,
B_t,
Z_t,
\pi_t,
Y_t,
C_t,
N_t,
w_t,
r_t,
i_t
\right).
\]
Здесь \(\Gamma_t\) -- распределение домохозяйств по индивидуальным состояниям, \(A_t\) и \(B_t\) -- агрегированные активы, \(Z_t\) -- агрегатные шоки, \(\pi_t\) -- инфляция, \(Y_t\) -- выпуск, \(C_t\) -- потребление, \(N_t\) -- занятость, \(w_t\) -- заработная плата, \(r_t\) -- реальная ставка, \(i_t\) -- номинальная ставка.

Домохозяйства решают динамическую задачу потребления и сбережений при неполных рынках. В минимальной записи индивидуальная задача имеет вид
\[
V_t(a,e)
=
\max_{c,a'}
\left\{
u(c)
+
\beta \mathbb E_t V_{t+1}(a',e')
\right\},
\]
при бюджетном ограничении
\[
c+a'
=
(1+r_t)a+w_t e+T_t,
\qquad
a'\geq \underline a.
\]
Распределение домохозяйств эволюционирует по закону
\[
\Gamma_{t+1}
=
\mathcal T(\Gamma_t;r_t,w_t,T_t,Z_t).
\]

Из полного состояния строятся распределительные статистики
\[
d_t=D(\Gamma_t).
\]
В основной постановке используются три статистики:
\[
d_t
=
\left(
\overline{\kappa}_t,
\ell_t,
\xi_t^r
\right),
\]
где \(\overline{\kappa}_t\) -- средняя предельная склонность к потреблению, \(\ell_t\) -- доля низколиквидных домохозяйств, \(\xi_t^r\) -- процентная экспозиция балансов.

Методологическая основа работы -- метод последовательностей. Равновесие HANK-модели записывается как система
\[
\mathcal F(\mathbf X,\mathbf i,\mathbf z)=0,
\]
где \(\mathbf X=(X_0,\ldots,X_T)\), \(\mathbf i=(i_0,\ldots,i_T)\), \(\mathbf z=(z_0,\ldots,z_T)\). Локально вокруг стационарного состояния равновесное отображение имеет вид
\[
d\mathbf X
=
\mathcal J_i d\mathbf i
+
\mathcal J_z d\mathbf z.
\]

Для информационного эксперимента используются не все компоненты полного состояния, а переменные, порождённые HANK-моделью:
\[
q_t
=
\left(
\pi_t,
y_t,
C_t,
\overline{\kappa}_t,
\ell_t,
\xi_t^r
\right).
\]
Здесь \(y_t\) -- разрыв выпуска, то есть переменная, входящая в стабилизационный критерий.
Их равновесная динамика порождается Jacobians метода последовательностей:
\[
d\mathbf q
=
\mathcal J_i^q d\mathbf i
+
\mathcal J_z^q d\mathbf z.
\]

Это принципиально отличает основную постановку от ручной редуцированной модели в пространстве состояний. Низкоразмерные объекты \(q_t\) и \(d_t\) являются статистиками и откликами HANK-экономики, а не самостоятельной экономической динамикой, заданной отдельно от HANK.
