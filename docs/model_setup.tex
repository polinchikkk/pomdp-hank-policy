\section{Постановка модели}

Полное состояние HANK-экономики обозначается через \(X_t^{HANK}\). Оно включает агрегированные переменные, распределение домохозяйств и экзогенные шоки. В вычислительной части работы используется редуцированное состояние
\[
x_t = R(X_t^{HANK}),
\]
которое состоит из агрегатного блока и распределительного блока:
\[
x_t =
\left(
\pi_t^{gap},
y_t^{gap},
r_t^*,
\overline{MPC}_t,
share_t^{liq}
\right).
\]

Центральный банк не наблюдает \(x_t\) напрямую. Он получает шумные наблюдения
\[
o_t = Mx_t + \nu_t.
\]

Информационное множество регулятора:
\[
\mathcal I_t =
\sigma(o_0,\ldots,o_t,i_0,\ldots,i_{t-1}).
\]

Правило политики строится по информационному состоянию
\[
s_t = \phi(\mathcal I_t),
\qquad
i_t = \mu(s_t).
\]
