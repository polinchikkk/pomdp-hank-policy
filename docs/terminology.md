# Термины проекта

Этот файл фиксирует словарь, который используется в статье и README.

## Экономические объекты

- `distributional_statistics` - распределительные статистики: средняя предельная склонность к
  потреблению, доля низколиквидных домохозяйств и чувствительность к ставке.
- `distributional_signals` - распределительные сигналы: шумные наблюдения или фильтрованные
  оценки распределительных статистик.
- `aggregate_state_estimate` - агрегатная оценка состояния по фильтру Калмана.
- `information_regime` - информационный режим регулятора: набор наблюдений и способ их сжатия в
  состояние, подаваемое в правило процентной ставки.
- `information_value` - предельная ценность информации, \(\Delta J\), то есть снижение
  оптимизированных потерь при переходе между информационными режимами.

## Расчётные блоки

- `feedback_rate_check` - проверка с обратной связью ставки.
- `lqg_benchmark` - линейно-квадратичный ориентир.
- `signal_content_checks` - проверки содержательности распределительного сигнала.
- `comparative_statics` - сравнительная статика ценности распределительной информации.

## Нежелательные термины в тексте статьи

В статье и README не используются как основные термины:

- `information ladder`;
- `closed-loop local projection`;
- `dashboard`;
- `oracle`;
- `protocol`;
- `placebo`;
- `policy loss`.

В коде эти имена могут сохраняться для совместимости со старыми файлами.
