# Исследовательский вопрос

Работа изучает ценность распределительной информации для денежно-кредитной политики при
неполной наблюдаемости в локальной HANK/SSJ-среде.

Главный вопрос:

> в каких условиях распределительные статистики домохозяйств имеют самостоятельную ценность для выбора ставки, если локальная динамика экономики задаётся HANK-моделью и методом последовательностей?

Главная линия:

```text
HANK-модель
→ равновесное отображение метода последовательностей
→ эксперимент с неполной информацией
→ ценность информации для политики
```

## Информационные состояния

В основной HANK/SSJ-версии сравниваются:

1. агрегатные наблюдения;
2. история агрегатных наблюдений;
3. оценённые агрегаты;
4. агрегаты и наблюдаемые распределительные сигналы;
5. оценённые агрегаты и распределительные статистики;
6. полная информация как верхняя граница.

## Рабочая гипотеза

Распределительная информация особенно полезна тогда, когда агрегатные показатели хуже отражают будущую силу трансмиссии ставки. Если хорошая фильтрация агрегатов уже почти исчерпывает релевантную информацию, предельная ценность распределительных статистик может быть малой.
