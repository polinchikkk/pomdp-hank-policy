# Исследовательский вопрос

Работа изучает, какая информация о HANK-экономике нужна центральному банку для выбора простой процентной ставки при неполной наблюдаемости.

Главный вопрос:

> имеют ли распределительные статистики, такие как средняя предельная склонность к потреблению и доля низколиквидных домохозяйств, самостоятельную ценность для правила денежно-кредитной политики сверх агрегатных показателей?

Работа не сравнивает алгоритмы управления. Основной объект сравнения — информационное состояние регулятора.

Иерархия информационных состояний:

1. только агрегаты;
2. оценённые агрегаты;
3. оценённые агрегаты и распределительные статистики;
4. полная информация как верхняя граница.
