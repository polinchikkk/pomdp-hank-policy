\subsection{Обоснование редуцированного представления состояния}

Используемое в работе редуцированное представление состояния не претендует на полное описание внутренней структуры HANK-модели. Его назначение скромнее и содержательнее: выделить те скрытые компоненты, которые имеют экономический смысл, поддаются восстановлению по наблюдаемым данным и важны для выбора процентной ставки.

Обоснование такого представления строится по трем линиям. Во-первых, каждая компонента состояния связана с одним из экономических каналов HANK-модели. Во-вторых, состояние проверяется на прогностическую достаточность: оно должно содержать информацию о будущей инфляции, разрыве выпуска и будущей функции потерь. В этой проверке используется только информация, доступная до выбора текущей ставки, включая лагированную ставку. В-третьих, состояние проверяется на достаточность для задачи политики: ранжирование правил, полученное в редуцированной постановке, должно сохраняться при проверке через HANK-переход.

Результаты показывают, что базовым представлением состояния в среднем следует считать макроэкономическое состояние с учётом скрытого режима. Распределительно расширенное состояние разумно трактовать как содержательное расширение, а не как универсально лучший набор. Главный прирост прогностического качества для будущей функции потерь связан именно с учётом скрытого режима.

Проверка через HANK-переход подтверждает устойчивость основного ранжирования правил. При этом её нужно трактовать аккуратно. Если допустимая амплитуда траектории меньше единицы, то траектория проходит через переходный решатель HANK только после уменьшения масштаба. Поэтому эта проверка не заменяет полную переоценку правил в исходной HANK-модели. Её смысл уже: проверить согласованность качественного вывода. В таблице \ref{tab:common_scale_pairwise_intervals} отдельно вынесено число сопоставимых траекторий. Для пары двух неклассических правил доступны все 12 отложенных траекторий, тогда как для сравнений с классическим правилом остаются лишь 1--3 траектории в зависимости от сценария. Поэтому сравнения с классическим правилом следует читать осторожнее.
