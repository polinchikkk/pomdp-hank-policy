# Обоснование редуцированного представления состояния

## Экономический смысл

Каждый компонент редуцированного состояния связан с отдельным каналом HANK-модели. Сводная таблица дана в файлах `state_component_interpretation.csv` и `table_state_component_interpretation.tex`.

## Прогностическая достаточность

Проверка прогноза строится только на той информации, которая известна до выбора текущей ставки. В признаки входит лишь лагированная ставка; текущая выбранная ставка не используется.

Главные итоги такие:

- для будущей функции потерь лучший средний результат даёт `Макроэкономическое состояние + скрытый режим` (`R^2 = 0.182`);
- для инфляции лучший средний результат даёт тот же набор (`R^2 = 0.784`);
- для разрыва выпуска лучшие средние результаты дают `Макроэкономическое состояние + скрытый режим` и `Макроэкономическое состояние` (`R^2 = 0.750`).

Главный смысл этой проверки не в том, что нужно механически добавлять как можно больше переменных. Наибольший прирост для будущей функции потерь даёт именно учёт скрытого режима.

Это особенно заметно в сценариях с бедной наблюдаемой информацией:

- при ограниченном наборе и умеренной разделимости режимов `R^2` растёт с `0.058` до `0.336`;
- при ограниченном наборе и высокой разделимости режимов `R^2` растёт с `0.053` до `0.288`.

Значит, скрытый режим здесь важнее, чем простое расширение набора распределительных переменных.

## Достаточность для задачи политики

Следующий вопрос состоит в том, сохраняется ли ранжирование правил при проверке через HANK-переход.

Главный результат:

- на средних траекториях знаки попарных сравнений совпадают в `12 из 12` случаев;
- при общей амплитуде траекторий `0.25` совпадение также равно `12 из 12`;
- на амплитудах `0.10` и `0.25` ранговый порядок правил сохраняется полностью.

Есть и важная оговорка. Для сравнений с классическим правилом сходятся не все траектории: на сценарий остаётся от `1` до `3` сопоставимых траекторий. Для сравнения двух неклассических правил доступны все `12` траекторий. Поэтому сравнения с классическим правилом надо понимать как локальную численную проверку, а не как полную статистическую переоценку.

## Итог

Редуцированное состояние не обязано полностью воспроизводить всю внутреннюю динамику HANK-модели. Для задачи политики достаточно более слабого и более важного свойства: оно должно сохранять информацию, нужную для прогноза будущих потерь, и давать то же качественное ранжирование правил.

Именно это текущая проверка и подтверждает:

1. базовым представлением состояния разумно считать `Макроэкономическое состояние + скрытый режим`;
2. `Распределительно расширенное состояние` лучше трактовать как содержательное расширение, а не как универсально лучший набор;
3. проверка через HANK-переход поддерживает основные выводы о сравнении правил.
