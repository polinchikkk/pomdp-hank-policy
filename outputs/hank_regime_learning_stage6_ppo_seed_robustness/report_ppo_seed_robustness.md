# Проверка устойчивости отрицательного результата для PPO

Проверка выполнена на трёх независимых запусках PPO: `11`, `22` и `33`.
В каждом запуске для каждого сценария выбирался лучший сохранённый шаг по проверочным траекториям `500`--`509`.
После этого правило оценивалось на `50` независимых траекториях `900`--`949`.

Сравнение проводится только с линейным правилом на том же самом входе.

## Главный вывод

Отрицательный результат для PPO устойчив.

- Ни в одном сценарии PPO не оказался лучше линейного правила в среднем.
- Во всех восьми сравнениях средняя разность `ΔJ = J(PPO) - J(линейное)` положительна.
- Неустойчивых траекторий нет.

Это означает, что слабый результат PPO нельзя объяснить одним неудачным запуском или численной неустойчивостью.

## По сценариям

- Базовый макроэкономический набор × умеренная разделимость режимов:
  лучший результат PPO всё равно хуже линейного правила и на узком входе, и на расширенном.
- Базовый макроэкономический набор × высокая разделимость режимов:
  PPO также хуже линейного правила во всех запусках.
- Ограниченный информационный набор × умеренная разделимость режимов:
  PPO иногда выигрывает на отдельных траекториях, но в среднем всё равно уступает.
- Ограниченный информационный набор × высокая разделимость режимов:
  отрицательный результат для PPO сохраняется.

## Что читать в таблице

- `Запуск` — номер запуска PPO.
- `Итерация` — выбранный шаг обучения.
- `Потеря на тесте` — средняя накопленная потеря на независимых траекториях.
- `ΔJ к линейному правилу` — насколько PPO хуже или лучше линейного правила на том же входе.
- `Доля побед` — доля траекторий, на которых PPO лучше линейного правила.
- `Неустойчивые траектории` — число траекторий с численной неустойчивостью.

Если `ΔJ` положительно, то PPO хуже линейного правила.

## Короткий итог для приложения

Проверка на трёх запусках подтверждает основной вывод этапа 6:

> в текущей постановке PPO не даёт самостоятельного выигрыша сверх сильного линейного правила на том же входе.
