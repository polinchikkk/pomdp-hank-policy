# Основная матрица сравнений этапа 6

## Что именно сравнивается

В основном тексте этапа 6 остаются четыре чисто монетарных варианта:

1. правило при полной информации;
2. классическое правило по оценённому состоянию;
3. отобранное правило по оценённому состоянию;
4. отобранное правило по наблюдаемым переменным.

Такая матрица позволяет раздельно посмотреть:

- что даёт оценивание скрытого состояния;
- что даёт отказ от жёстко заданного правила;
- насколько велик разрыв до варианта с полной информацией.

## Главный вывод

Отобранное правило по оценённому состоянию заметно лучше классического правила по оценённому состоянию. По четырём основным сценариям накопленная функция потерь снижается на `56.7--71.1\%`.

Но этот результат не надо читать как чистый выигрыш от «сложной архитектуры». Здесь одновременно меняются:

- жёсткость самого правила;
- способ выбора коэффициентов;
- согласованность правила с доступной информацией.

Поэтому корректная формулировка проще:

> отобранное правило заметно лучше узкого фиксированного правила.

## Когда особенно важна фильтрация

Ценность оценивания скрытого состояния зависит от информационного режима.

- При базовом макроэкономическом наборе правило по наблюдаемым переменным и правило по оценённому состоянию близки.
- При ограниченном информационном наборе правило по оценённому состоянию становится заметно лучше.

Значит, явное восстановление скрытого состояния особенно полезно тогда, когда наблюдаемой информации мало.

## Как это связывается с полной HANK-моделью

Полная HANK-модель здесь выступает как содержательная экономическая среда. Сравнение самих правил проводится на редуцированном представлении состояния. Поэтому этот блок надо понимать как сравнение способов принятия решений при неполной наблюдаемости, а не как полную переоптимизацию политики внутри исходной HANK-модели.

Проверка через HANK-переход используется отдельно как внешняя проверка согласованности вывода.
