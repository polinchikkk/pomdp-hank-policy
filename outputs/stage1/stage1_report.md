# Базовый бейзлайн на RBC-модели

## Постановка модели

На первом этапе реализована стандартная модель реального делового цикла (RBC) с персистентным технологическим шоком.

- **Базовая модель:** RBC-модель с технологическим шоком.
- **Переменные состояния:** логарифмическое отклонение капитала от стационарного уровня и уровень технологии `z_t`.
- **Переменные выбора:** логарифмические отклонения потребления и труда.
- **Закон движения шока:** `z_(t+1) = rho z_t + sigma epsilon_(t+1)`.

Цель этого этапа:
- корректность задания модели
- получение решения стандартным методом
- симуляцию траекторий
- построение импульсных откликов
- базовые проверки

---

## Метод решения

Решение модели строится как **локальная аппроксимация первого порядка** в окрестности детерминированного стационарного состояния

Используемая вычислительная схема:
- стационарное состояние вычисляется в замкнутой форме;
- якобианы равновесных условий находятся численно с помощью конечных разностей;
- на основе линеаризованной системы восстанавливаются матрицы политики и перехода.

Таким образом, этап 1 представляет собой **классический локальный baseline**, который в дальнейшем будет использоваться как точка отсчёта для более сложных постановок.

## 4. Калибровка и стационарное состояние

Использовалась следующая калибровка параметров:

- `beta = 0.990`
- `alpha = 0.360`
- `delta = 0.025`
- `rho = 0.950`
- `sigma = 0.010`

Из условий стационарного состояния получены:

- параметр дезполезности труда `psi = 7.903`
- выпуск `y = 1.2223`
- потребление `c = 0.9089`
- инвестиции `i = 0.3134`
- капитал `k = 12.5365`
- труд `n = 0.3300`

Полученные значения выглядят согласованными со стандартной RBC-калибровкой и используются как базовая точка для локальной аппроксимации.

## 5. Диагностические проверки

### 5.1. Качество линейного решения

- максимальный остаток в линеаризованной системе: `8.826e-15`
- спектральный радиус матрицы перехода: `0.9575`

Это означает, что линейная система решена с очень высокой точностью, локальная динамика устойчива, так как спектральный радиус меньше единицы.

### 5.2. Качество на стохастической симуляции

- максимальный условный остаток: `0.0030`
- RMS условных остатков: `0.0004`
- время выполнения: `0.279` сек.

Условные остатки остаются малыми в окрестности стационарного состояния, что согласуется с использованием локальной аппроксимации первого порядка.

### 5.3. Проверка знаков IRF

После положительного технологического шока на ударе наблюдается:

- выпуск увеличивается;
- потребление увеличивается;
- инвестиции увеличиваются;
- труд увеличивается.

Это соответствует стандартной экономической логике RBC-модели.

### 5.4. Устойчивость к выбору seed

Дополнительно проверена устойчивость симуляции по нескольким значениям `seed`.

- `seed = 0`: конечные траектории есть, max |state dev| = `0.0921`, max |residual| = `0.0034`
- `seed = 1`: конечные траектории есть, max |state dev| = `0.1097`, max |residual| = `0.0023`
- `seed = 2`: конечные траектории есть, max |state dev| = `0.1040`, max |residual| = `0.0027`
- `seed = 3`: конечные траектории есть, max |state dev| = `0.0903`, max |residual| = `0.0040`
- `seed = 4`: конечные траектории есть, max |state dev| = `0.1166`, max |residual| = `0.0037`

По всем протестированным seed’ам траектории остаются конечными, а ошибки — малыми, что подтверждает устойчивость baseline-реализации.

## 6. Интерпретация результатов

Полученные результаты позволяют сделать следующие выводы.

1. **Импульсные отклики имеют экономически осмысленную форму.**  
   Положительный технологический шок приводит к росту выпуска, потребления, инвестиций и труда на ударе, что соответствует стандартным свойствам RBC-модели.

2. **Локальная динамика устойчива.**  
   Спектральный радиус матрицы перехода меньше единицы, следовательно, траектории не расходятся.

3. **Локальная аппроксимация работает корректно в окрестности стационарного состояния.**  
   Условные остатки на симуляции малы, что говорит о приемлемом качестве baseline-решения для целей первого этапа.
