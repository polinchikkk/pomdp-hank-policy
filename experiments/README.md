# Расчётные скрипты

В этой папке оставлены только финальные точки входа и архив исторических расчётов.

- `final/` -- короткие обёртки для воспроизведения работы.
- `archive/` -- ранние и вспомогательные `expXX`-скрипты, сохранённые для воспроизводимости истории проекта.

Основной путь запуска описан в `README.md` в корне репозитория и в `docs/reproducibility.md`.
